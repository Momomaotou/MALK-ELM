# Code Description
 路MFKL. py is the running file of our proposed algorithm.  
 路The NSL-KDD dataset is the default dataset.  
 路If you want to change the dataset, you need to manually adjust the size of the last Dense layer of the model to match the number of labels in the dataset.  
 路Select the dataset for feature grouping through the FeatureGroup.py. Then write the grouping results into the FeatureRead.py.
